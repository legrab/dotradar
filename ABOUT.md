# DOTRADAR

## Opinionated repository evaluation tool for .NET solutions

**Deep look, big picture**

DOTRADAR provides a **quick and efficient check of the overall health of a .NET solution**
by quantifying trustworthiness and reliability of a system via metrics regarding
complexity, standardization, coverage, and other best practices.
It delivers a transparent, broad evaluation to determine upkeep costs and implementation velocity.
